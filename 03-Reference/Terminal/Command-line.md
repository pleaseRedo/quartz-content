# Terminal Command Line Navigation Shortcuts

## Move Cursor

| Shortcut | Function |
|---|---|
| `Ctrl + A` | Move cursor to the beginning of the command line |
| `Ctrl + E` | Move cursor to the end of the command line |
| `Alt + B` | Move cursor backward by one word |
| `Alt + F` | Move cursor forward by one word |
| `Ctrl + ←` | Move cursor backward by one word, supported by some terminals |
| `Ctrl + →` | Move cursor forward by one word, supported by some terminals |

## Delete Text

| Shortcut | Function |
|---|---|
| `Ctrl + U` | Delete everything before the cursor |
| `Ctrl + K` | Delete everything after the cursor |
| `Ctrl + W` | Delete one word before the cursor |
| `Alt + D` | Delete one word after the cursor |

## Most Useful Shortcuts

```bash
Ctrl + A    # Go to the beginning of the command
Ctrl + E    # Go to the end of the command
Ctrl + U    # Delete everything before the cursor
Ctrl + K    # Delete everything after the cursor
